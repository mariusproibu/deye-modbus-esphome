# deye-modbus-esphome

## Deye 3 phase Low voltage inverter ##

Communication with Deye Inverter using Home Assistant's esphome & RS485 protocol communication. Based on the work of Lewa-Reka (https://github.com/Lewa-Reka/esphome-deye-inverter)
For details look in the original repository. This repo is ment to collect modifications from my side and provide some kind of a back-up to make shore that no brake-in changes in original repo will bring down my inverter
